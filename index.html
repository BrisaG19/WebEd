Welcome to WebEd

On this site you will learn how to use HTML and CSS

To begin, most websites use html, head, body, and paragraph tags. HTML is short for Hypertext Markup Language. HTML is used for creating websites and web applications. CSS is short for Cascading Style Sheets which is used for editing websites. HTML and CSS work closely together to create a website how a person wants it.

Using these elements allows you to insert images and align text as well as adding color to your website. Website creators have immense knowledge on these topics and manage to create astounding websites.
